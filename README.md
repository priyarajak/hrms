## ✨ Project Overview

HRMS Lite is a production-ready web application that allows an admin to:

• Manage employee records  
• Track and monitor daily attendance  
• Filter attendance by date range  
• View attendance insights and summaries  

The application demonstrates **end-to-end full-stack development**, clean UI design, and deployment readiness.

---

## 🖥️ Application Preview

### Core Features

👨‍💼 Employee Management  
- Add employees  
- Delete employees  
- Unique employee validation  
- Email validation  

📅 Attendance Management  
- Mark daily attendance

- Filter by date range  
- Attendance summary  
- Sorted newest → oldest records  

🎨 Professional UI  
- Responsive dashboard layout  
- Toast notifications  
- Smooth page transitions  
- Empty states & loading states  
- Confirmation dialogs

## 🛠️ Tech Stack

### 🔹 Frontend
- React (Vite)
- Tailwind CSS
- React Router
- Framer Motion (page transitions)
- React Hot Toast (notifications)
- Axios (API integration)

### 🔹 Backend
- FastAPI (Python)
- Motor (Async MongoDB driver)
- Pydantic (validation)
- Uvicorn (ASGI server)

### 🔹 Database
- MongoDB Atlas (Cloud)

### 🔹 Deployment
| Layer | Platform |
|---|---|
| Frontend | Vercel |
| Backend | Render |
| Database | MongoDB Atlas |
| Version Control | GitHub |
